# Portfolio

A modern, interactive portfolio built with **React**, **Vite**, **GSAP**, and **Three.js**, and deployed using **Cloudflare Workers**.

---

## Overview
This project showcases selected work with smooth animations, engaging 3D visuals, and responsive UI. It’s designed to feel fast, polished, and easy to explore—while keeping performance in mind.

---

## Features
- GSAP-powered scroll and reveal animations  
- Three.js background/visuals for an immersive feel  
- Fast, component-driven React architecture  
- Clean section structure for easy updates  
- Optimized assets for quick loading

---

## Tech Stack
- React  
- Vite  
- GSAP  
- Three.js  
- Cloudflare Workers  

---

## Installation
```bash
npm install
npm run dev
```

---

## Deployment
Deployed via **Cloudflare Workers** using the project’s build output.

```bash
npm run build
```

(Deploy the `dist/` output to your Worker/hosting setup.)
